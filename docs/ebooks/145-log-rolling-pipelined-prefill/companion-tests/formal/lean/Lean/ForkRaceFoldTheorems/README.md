- Parent README: [../README.md](../README.md)

This subdirectory covers the in-tree constructive theorem package. The shared compiler-emitted kernel theorems referenced by `THM-GNOSIS-*`, including the bounded affine queue-family `continuousHarris` witness package and the bounded coupled-kernel tethering result for inter-app handoff pressure, live in `open-source/gnosis/GnosisProofs.lean` rather than in this subdirectory.
