# Topology Tests

Parent: [../README.md](../README.md)

Focused tests for topology analysis, projection contracts, and AEON object envelopes.

## Files

- `formal-claims.test.ts`: validates named topology formulas and protocol claims.
- `gnosis-impossible-systems.test.ts`: validates the impossible-system catalog.
- `object-envelope.test.ts`: verifies that one `aeon-object/v1` envelope can expose document, app, agent, and stream projections.
- `topology.test.ts`: validates analyzer and sampler behavior.
